# 개인정보처리방침 (Privacy Policy)

**최종 업데이트: 2026년 6월**

---

## 1. 개요

본 웹사이트 **원빌리언달러 도네이션** (이하 "사이트")은 개인적인 투자 철학과 자산 설계 아카이브를 공유하는 비상업적 개인 사이트입니다.

---

## 2. 수집하는 정보

**본 사이트는 방문자의 개인정보를 일체 수집하지 않습니다.**

- ❌ 쿠키(Cookie) 미사용
- ❌ 트래킹 픽셀 미사용
- ❌ 분석 도구(Google Analytics 등) 미사용
- ❌ 회원가입·로그인 기능 없음
- ❌ 광고 네트워크 미연결

---

## 3. 제3자 서비스

본 사이트는 **GitHub Pages**를 통해 호스팅됩니다.  
GitHub의 서버 접속 로그(IP 주소, 접속 시간 등)는 GitHub의 자체 정책에 따라 처리될 수 있습니다.

- GitHub 개인정보처리방침: [https://docs.github.com/ko/site-policy/privacy-policies/github-general-privacy-statement](https://docs.github.com/ko/site-policy/privacy-policies/github-general-privacy-statement)

## 4. 외부 링크

본 사이트에는 외부 사이트로의 링크가 포함될 수 있습니다.  
해당 외부 사이트의 개인정보 처리에 대해서는 본 사이트가 책임지지 않습니다.

본 개인정보처리방침은 사이트 운영 방식이 변경될 경우 업데이트될 수 있으며, 페이지 상단의 날짜를 통해 최신 버전을 확인할 수 있습니다.
