# 저작권·콘텐츠 이용 안내

이 사이트의 글, 직접 만든 도표, 설명 구조와 데이터 편집물은 별도 표시가 없는 한 사이트 운영자가 작성·구성한 저작물입니다. 외부 자료와 이미지의 권리는 각 원저작자에게 있으며, 출처가 표시된 자료는 해당 권리자의 이용 조건을 따릅니다.

## 가능한 이용

- 원문 페이지로 연결되는 **링크 공유**
- 출처와 링크를 밝힌 짧은 범위의 인용
- 개인적 학습을 위한 페이지 열람과 메모

## 사전 허가가 필요한 이용

- 글이나 표의 전부 또는 상당 부분을 다른 사이트·뉴스레터·영상·데이터셋에 복제
- 자동 수집한 콘텐츠의 재배포, 번역본 공개 또는 상업적 이용
- 사이트가 만든 이미지·차트·데이터 구성을 출처 없이 다시 게시

허가가 필요하면 이용할 페이지, 범위, 매체와 목적을 [이메일](/contact)로 알려주세요.

## 선택·복사 제한

본문의 텍스트 선택과 일반적인 복사 동작은 무단 복제를 줄이기 위한 운영 정책으로 제한되어 있습니다. 링크, 표, 코드 등 확인과 활용이 필요한 요소는 일부 선택할 수 있습니다. 이 기술적 제한은 저작권의 범위를 넓히거나 법이 허용하는 정당한 인용을 막기 위한 법률적 판단이 아닙니다.
