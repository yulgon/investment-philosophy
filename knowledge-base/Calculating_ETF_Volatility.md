# ETF의 변동성(Volatility) 계산 방법

퀀트 투자와 리스크 패리티(Risk Parity) 모델에서 **변동성(Volatility)**은 가장 중요한 입력값입니다. 금융공학에서 변동성은 주로 **'수익률의 표준편차(Standard Deviation)'**를 의미합니다.

변동성을 계산하는 과정은 크게 3단계로 나뉘며, 엑셀이나 파이썬으로 쉽게 구현할 수 있습니다.

---

## 1. 변동성 계산의 3단계 수학적 원리

### Step 1: 일간 수익률(Daily Return) 계산
특정 기간(예: 최근 60일, 120일, 1년) 동안의 매일매일의 수익률을 구합니다.
- **수식**: $R_t = \frac{P_t - P_{t-1}}{P_{t-1}}$
  *(여기서 $P_t$는 오늘 종가, $P_{t-1}$은 어제 종가입니다.)*
- 더 정교한 계산을 위해서는 자연로그를 이용한 로그 수익률($\ln(P_t / P_{t-1})$)을 사용하기도 합니다.

### Step 2: 일간 수익률의 표준편차(Standard Deviation) 산출
Step 1에서 구한 일간 수익률 데이터들의 평균을 구하고, 그 평균으로부터 매일의 수익률이 얼마나 흩어져 있는지(분산)를 구한 뒤 루트를 씌워 **일간 표준편차**를 구합니다.

### Step 3: 연환산 변동성(Annualized Volatility)으로 변환 ⭐️
일간 변동성만으로는 직관적인 비교가 어렵기 때문에, 이를 '1년 단위'로 환산합니다. 주식 시장은 1년에 약 252일 열리므로, **일간 표준편차에 $\sqrt{252}$ (약 15.87)를 곱해줍니다.**
- **수식**: $Annual Volatility = \sigma_{daily} \times \sqrt{252}$
- 만약 주간(Weekly) 데이터로 구했다면 $\sqrt{52}$, 월간(Monthly) 데이터라면 $\sqrt{12}$를 곱합니다.

## 2. 실전 계산 방법

### A. 엑셀(Excel) 또는 구글 스프레드시트 활용
가장 직관적으로 계산할 수 있는 방법입니다.
1. Yahoo Finance 등에서 ETF의 과거 종가(Adjusted Close) 데이터를 다운로드합니다.
2. B열에 종가가 있다면, C열에 일간 수익률을 구합니다. `=(B3-B2)/B2`
3. 일간 표준편차를 구합니다. `=STDEV.P(C열 데이터 범위)`
4. 연환산 변동성을 구합니다. `=일간표준편차 * SQRT(252)`

### B. 파이썬(Python) 활용 (자동화)
퀀트 시스템을 구축할 때 가장 많이 쓰이는 방식입니다. `pandas` 라이브러리를 사용하면 단 3줄로 끝납니다.

```python
import yfinance as yf
import numpy as np

# 1. 데이터 다운로드 (예: VOO의 최근 1년 데이터)
data = yf.download('VOO', period='1y')

# 2. 일간 수익률 계산
daily_returns = data['Adj Close'].pct_change().dropna()

# 3. 연환산 변동성 계산
annual_volatility = daily_returns.std() * np.sqrt(252)

print(f"VOO의 연환산 변동성: {annual_volatility * 100:.2f}%")
```

### C. 직접 계산하지 않고 확인하는 방법 (무료 툴)
직접 계산이 번거로울 때는 다음 사이트들을 활용할 수 있습니다.
- **Portfolio Visualizer (포트폴리오 비주얼라이저)**: 백테스트 툴에서 자산을 입력하면 `Stdev (Standard Deviation)` 항목에 연환산 변동성이 바로 나옵니다.
- **TradingView (트레이딩뷰)**: 지표(Indicators)에서 `Historical Volatility (역사적 변동성)`를 검색하여 차트에 추가하면 현재 시점의 변동성을 퍼센트로 볼 수 있습니다.

## 3. 리스크 패리티 적용 시 주의사항 (Look-back Period)

변동성을 계산할 때 **'과거 며칠의 데이터를 볼 것인가(Look-back Period)'**가 매우 중요합니다.
- **최근 20일 (약 1개월)**: 시장의 최근 충격을 매우 빠르게 반영하지만 비중이 너무 자주 바뀝니다(노이즈가 심함).
- **최근 252일 (1년)**: 장기적인 추세를 잘 반영하지만, 갑작스러운 코로나 하락장 같은 폭락을 너무 늦게 반영합니다.
- **추천**: 보통 퀀트에서는 **60일(약 3개월)** 또는 **120일(약 6개월)**의 단기와 중기 변동성을 혼합하여 사용하는 경우가 많습니다. 혹은 지수 가중 이동평균(EWMA) 모델을 사용하여 최근 데이터에 더 큰 가중치를 두어 변동성을 계산하기도 합니다.
